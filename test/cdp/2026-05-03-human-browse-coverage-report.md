## 五、CDP 人类浏览技术要点

1. **Tab 复用**: 预先分配 blank tab，避免频繁 `PUT /json/new`（Chrome 有频率限制）
2. **点击后等待**: SPA 页面渲染需要等待，用 `setTimeout(r, 5000)` 在每次点击后等 5 秒
3. **滚动模拟**: `window.scrollBy(0, 600)` × 3 次，触发懒加载内容
4. **选择器策略**: `a[href*="stock"]` 比 `[class*="tab"]` 更精准（泛 class 容易误触）
5. **数据提取**: `document.body.innerText` 截取前 5000 字符，比抓 API 更稳（SPA 数据都在 DOM 里）
6. **SPA hash 路由**: 东财 `gridlist.html#hs_a_board` 的 hash 部分不会触发 `Page.navigate`，需从页面内点击触发
