## 关键发现

### highest_board（连板高度）✅ 直接可获取
- **来源**: `line_lst` 的最新记录 `[最高连板天数]`
- **最新值** (20260430): **4连板**（越剑智能）
- **历史最高**: **18连板**（锋龙股份 sz002931，20260123）
- **获取方式**: 无需 CDP，curl 获取 HTML 后解析 `line_lst[0]`

```python
import re, json, subprocess
html = subprocess.run(['curl', '-s', 'https://zhangtingke.com/zt_lbgd_line'], capture_output=True).stdout
data = json.loads(re.search(r'let lbgd_dict = (\{.+\});', html).group(1))
highest_board = data['line_lst'][0][1]  # 最新最高连板天数
```

### break_board_rate（炸板率）⚠️ 无直接字段，需计算
- 该页面没有炸板率直接字段
- akshare `stock_zt_pool_zbgc_em` 有"炸板次数"列，但无炸板率
- **建议**: 炸板率 = 炸板次数 / (涨停数 + 炸板数) × 100%

## 与 akshare 的关系

akshare 连板池 `stock_zt_pool_zbgc_em`（14条）：
- 仅包含当前正在连板的股票
- "涨停统计=X/Y"是**历史累计**连板数，非当日连板高度
- **缺少当前精确连板天数**

涨停客 `lbgd_lst`：
- 每日所有连板股票（含历史）
- **第3列 = 当前连板天数**（精确值）
- 覆盖所有历史连板记录（不只是当前连板）
